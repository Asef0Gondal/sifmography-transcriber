# 🎙 Sifmography Transcriber

A premium macOS-native audio transcription tool utilizing MLX Whisper community models. Converts high-volume voiceovers and audio reference assets into perfectly timed transcripts.

---

## ✨ Features
- **⚡ 100% Local Inference:** Utilizing `mlx-whisper` for near-instant transcription.
- **🕓 Dynamic SRT/VTT:** Exports precise subtitle alignments automatically.
- **💻 Minimalist Launcher:** Includes easy double-click `launch_transcribe.command` support.
